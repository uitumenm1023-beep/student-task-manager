**Portfolio Website**

This project is my personal portfolio website. This website will show all the details about me, experience of real world projects(skills), also different ways to contact me.



**Version 1 (v1.0)**

What users can see firstly?

Users can:
- View my introduction
- Learn about my experience
- See information about my projects
- Contact me through provided links

- Homepage
- Introduction section
- About me section
- Experience section
- Contact section

**Navigation**
- Navigation bar
- Links to different sections (About, Experience, Contact)

**Theme Mode**
